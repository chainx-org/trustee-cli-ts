# trustee-tools CLI

A CLI for trustee-tools.



## Init Environment

```shell
$ yarn
$ yarn run build
$ yarn run link
```

## 查看信托信息

```
trustee-tools info
```

## 查看待提现列表

```
trustee-tools list
```

## 查看代签列表

```
trustee-tools tx
```

## 构造提现交易原文并提交上链

```
trustee-tools  create submit
```

## 构造提现交易原文不上链

```
trustee-tools  create
```

## 响应待签原文并提交上链

```
trustee-tools  respond submit

```
