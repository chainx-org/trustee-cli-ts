# trustee-tools CLI

A CLI for trustee-tools.



## Publishing to NPM

To package your CLI up for NPM, do this:

```shell
$ npm login
$ npm whoami
$ npm lint
```

## 查看信托信息

```
trustee-tools info
```

## 查看待提现列表

```
trustee-tools list

```

## 查看代签列表

```
trustee-tools tx
```
